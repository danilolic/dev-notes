Colinha do GIT

Para inicar uma pasta com o GIT: Vai até a pasta que você quer inicar com o GIT clica com o botão direito do mouse e seleciona Git bash here igual na foto.

Depois disso é só digitar: git init pronto essa pasta funciona os comandos git

Você precisa configurar duas coisas no GIT, seu nome e seu email

Nome: git config --global user.name "Fulano de Tal"

email: git config --global user.email fulanodetal@exemplo.br

Quando você altera um arquivo e você quer commitar ele ou seja criar uma versão nova dele, seja versão 2 ou 3 ou qualquer número você precisa usar o comando git add nome_do_arquivo. Então seu eu criei o arquivo danilo.html preciso usar o comando git add danilo.html e para finalizar o commit preciso de uma mensagem, pode ser algo do tipo git commit -m'criei o arquivo chamado danilo.html'. Pronto você comitou o arquivo danilo.html, se você alterar o arquivo danilo.html colocando por exemplo mais um parágrafo, você precisa repetir os mesmos passos que seria git add danilo.html e depois o commit git commit -m'adicionei um parágrafo no arquivo danilo.html', você pode colocar qualquer mensagem, mas use uma mensagem que descreva o que você fez.

GIT com Github - Compartilhando seus projetos com o Mundo

O Github é uma rede social de programadores onde você pode compartilhar seus projetos com outras pessoas e você pode também ajudar outras pessoas nos projetos delas. Olha só que legal. Tudo que você precisa é criar uma conta no Github e é completamente de graça. Depois de cadastrar me sigam lá: https://github.com/danilolic.

No seu cadastro use o email que você configurou lá no:
git config --global user.email fulanodetal@exemplo.br

Depois do cadastro você está livre para baixar qualquer projeto do Github, você consegue isso usando o comando git clone só para testar baixem esse daqui:
git clone https://github.com/danilolic/dev-notes.git só toma cuidado que ele vai baixar dentro da pasta que o seu git bash here foi aberto. Se você quiser salva em outra pasta tem que abrir o git bash here em outra pasta.
